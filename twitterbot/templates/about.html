Inspired by Pentametron and H6O6, I made a robot that writes poetry. In particular, it tries to write limericks and sometimes it succeeds.

It works by listening to the public Twitter streaming API and trying to count the syllables of every tweet using the Python NLTK module. If that works, the tweet is stored in a SQL database and poems are generating by running a query on that database. If you don't know what these words mean, don't worry. If you do, check out the code on GitHub.
